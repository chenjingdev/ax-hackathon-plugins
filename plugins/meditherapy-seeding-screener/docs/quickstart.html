Seeding Screener

메디테라피meditherapy-seeding-screenerCodex 플러그인

인플루언서 시딩 후보의 공개 콘텐츠를 한국 동료심사 근거 5차원으로 채점해, 팔로워 수가 아니라 구매유발 결로 랭킹한다. 모든 점수는 원문 인용으로 방어한다(glass-box).

30초 시작

  1. 설치: 
    codex plugin marketplace add https://github.com/chenjingdev/ax-hackathon-plugins
codex plugin add meditherapy-seeding-screener@ax-hackathon-plugins
  2. Codex 채팅에 붙여넣기 (앱이면 "사용해 보기" 클릭): 
    번들 샘플 sample/input.kr.json으로 시딩 스크리닝을 시연해줘
  3. 팔로워 89만 메가가 적합도 13으로 3위, 팔로워 8.6천 더마 전문가가 78로 1위 — 역전 랭킹이 근거 인용과 함께 나오면 정상.

무엇을 하나

  • 5차원 스코어링 — 전문성·준사회성·정보성·후기·진정성. 가중치는 한국 동료심사 논문 근거로 동결(ontology.json).
  • glass-box 근거 — 모든 차원 점수에 원문 verbatim 인용을 붙인다. 인용은 실재해야 한다(환각 0 게이트).
  • 재스캔 — 온톨로지 갱신(v0.1→v0.2) 시 기존 후보를 재채점하고 순위 변화를 추적.
  • 산출 — 랭킹 숏리스트 + 후보별 dossier + 콘텐츠 보완 제안 + 사유 메모.

출력

  • 랭킹 표(purchase_intent_fit 0~100) + 후보별 dossier(차원 점수·confidence·원문 인용·출처 논문).
  • 점수가 아니라 근거가 당락을 가른다 — 마케터는 「2주간 직접 써보고 말씀드릴게요」 같은 실재 인용을 보고 결정한다.

하지 않는 것

  • 매출 귀인·바이럴 예측·진위 감사 아님 — 적합도는 추정이고 확정은 사람이 한다(reviewed_by=human-required).
  • 스크래핑 없음 — 콘텐츠는 입력으로 받는다. 자동 발송·확정 0.
